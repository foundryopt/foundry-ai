# Project Scope

A "Project" includes the full lifecycle:

- Development
- Design — see [[SHB Studio]]
- Construction / CA — see [[SHB Inc]]
- Procurement — see [[Builiq Inc]]
- Marketing
- Sales / Showroom
- Fund / Investors
- Property Management
- Warranty
- Executive / Decisions — escalation to [[SHB Group]]

All functions use **Email and Slack for intake**. See [[Interface Strategy]].
