# Warranty Closeout Checklist

## Overview

Standardized checklist completed before a warranty period expires to ensure all defects have been identified, claimed, and resolved — and that the transition from warranty-covered to owner-responsible maintenance is documented.

## When to Use

Complete this checklist for each warranty category (general, roofing, mechanical, etc.) at least 30 days before the warranty expiration date. One checklist per warranty category per project.

## Usage Notes

- Copy this checklist for each warranty category at each project.
- File the completed checklist in the project warranty records.
- Cross-reference with `property-management/warranty/warranty-claim-log-schema.md` to confirm all claims are accounted for.
- Items with manufacturer warranties extending beyond the contractor warranty should be tracked separately in the warranty inventory.
