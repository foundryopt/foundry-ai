# Entities

This directory holds **only overrides and compliance-specific differences** for individual legal entities.

## Entities

- `shb-group/`
- `shb-studio/`
- `shb-inc/`
- `builiq-inc/`
- `foundry-rooms/`
- `foundry-fund/`

## Policy

80-90% of all assets are shared across entities and belong in functional verticals (`development/`, `design/`, etc.).

Place an asset here **only** if it genuinely differs per entity — for example:
- Legal compliance documents
- Entity-specific regulatory SOPs
- Brand-specific deviations from shared processes

**Do not** duplicate shared assets into entity folders.
