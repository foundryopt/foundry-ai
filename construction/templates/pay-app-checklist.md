# Pay Application Checklist

## Overview

Standardized checklist for reviewing contractor pay applications before approval. One checklist per pay application per pay period. Supports the controls defined in `construction/sops/ops-pay-app-and-invoice-approval.md`.

## Pay Application Header

| Field | Value |
|---|---|
| Project | [Project name or code] |
| Pay period | [MM/YYYY or date range] |
| Applicant | [Contractor / sub name] |
| Pay-app number | [Sequential, e.g. `PA-001`] |
| Contract sum (current) | [Original + all executed COs] |
| Amount requested this period | [Dollar amount] |
| Retainage held this period | [Dollar amount] |
| Net amount due | [Dollar amount] |

## Verification Checklist

| # | Check | Pass / Fail | Notes |
|---|---|---|---|
| 1 | Schedule of values matches current contract (original + executed COs) | | |
| 2 | No line items reference unapproved PCOs or unexecuted COs | | |
| 3 | Contract sum on pay-app matches project accounting record | | |
| 4 | Percent-complete per line item verified by field walk | | |
| 5 | Stored material quantities verified on site and match pay-app | | |
| 6 | Stored materials are properly protected per manufacturer requirements | | |
| 7 | Retainage calculated correctly per contract terms | | |
| 8 | Previous payment amounts reconcile with accounting records | | |
| 9 | Conditional lien waiver received for this period (progress payment) | | |
| 10 | Unconditional lien waiver received for prior period | | |
| 11 | No outstanding back-charges or deductions unaccounted for | | |
| 12 | Insurance and bonding current and on file | | |

## Usage Notes

- Copy this template for each pay application review.
- Fill in all bracketed placeholders.
- Attach the completed checklist to the pay application before routing for authorization.
- File the signed checklist in the project financial records.
