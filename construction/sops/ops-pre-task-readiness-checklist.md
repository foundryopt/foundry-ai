# Pre-Task Readiness Checklist

## Purpose

Ensure every construction task has all prerequisites confirmed before field work begins — preventing rework, safety incidents, and schedule delays caused by incomplete preparation.

## Scope

All active construction projects across all entities. Required before any new phase, trade mobilization, or critical-path activity. Applies to project managers, site superintendents, and subcontractor foremen.

## Control: No Mobilization Without Signed Checklist

**No trade or crew may mobilize for a task until the pre-task readiness checklist is completed and signed by the site superintendent.**

- An incomplete checklist blocks the start of work.
- If a checklist item cannot be satisfied, the site superintendent must document the exception, obtain project manager approval, and note the item as "waived — with conditions" including mitigation measures.
- Any incident or delay traced to a waived item is flagged in the project incident log.

## Procedure

1. **Identify upcoming task.** Project manager or superintendent identifies the next task on the look-ahead schedule (minimum 1 week out for standard tasks, 2 weeks for critical-path).
2. **Initiate checklist.** Superintendent creates a pre-task readiness checklist for the task using the checklist categories below.
3. **Verify each category.** Superintendent confirms each item with the responsible party.
4. **Decision gate: Ready or not ready.**
   - **Ready** — All items confirmed or formally waived. Superintendent signs and files the checklist. Work may begin.
   - **Not ready** — Superintendent logs the blocking items, notifies the project manager, and reschedules or resolves.
5. **Conduct pre-task briefing.** Superintendent briefs the crew on scope, safety, logistics, and any waived items before work starts.
6. **File checklist.** Completed checklist is stored in the project record and referenced in the daily log.

## Checklist Categories

| Category | Verification Items |
|---|---|
| **Drawings & specs** | Current issued-for-construction drawings on site; relevant RFIs resolved; any pending ASIs or bulletins distributed |
| **Permits & inspections** | Required permits posted; prior-phase inspections passed and signed off |
| **Materials & equipment** | Materials delivered and inspected; equipment on site and operational; long-lead items confirmed with procurement |
| **Subcontractor readiness** | Sub mobilization confirmed; insurance and safety certs current; sub foreman briefed on scope |
| **Site conditions** | Work area accessible and clear; utilities located and marked; weather forecast reviewed for impact |
| **Safety** | Task-specific hazard analysis complete; required PPE identified and available; emergency procedures posted |
| **Open RFIs / COs** | No unresolved critical or high-priority RFIs affecting this task; no pending change orders affecting scope |
| **Coordination** | No conflicts with adjacent trades; crane/hoist schedule confirmed if applicable; MEP/structural coordination complete |

## Decision Gates

| Gate | Condition to Pass | Authority |
|---|---|---|
| Checklist initiated | Task appears on look-ahead schedule with defined scope | Project manager |
| Category sign-off | Each category verified or formally waived | Site superintendent |
| Readiness approval | All categories complete; checklist signed | Site superintendent |
| Waiver approval | Exception documented with mitigation measures | Project manager |
