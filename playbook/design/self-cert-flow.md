# Self-Certification Permit Flow

## Overview
Kuan holds a Self-Certification Architect designation with the City of Chicago.
This allows bypassing the standard plan review queue — saving 3–6 months per project.

## Process
1. **CDs complete** — full construction document set ready
2. **Self-cert review** — Kuan reviews against Chicago Building Code
3. **Application submission** — file with Chicago DOB via E-Plan
4. **Permit issuance** — typically within days (vs. months for standard review)

## Requirements
- Licensed Self-Certification Architect must stamp and sign
- All code compliance is architect's responsibility (no city reviewer backstop)
- Higher liability standard — design must be bulletproof
- Annual audit by city (random project selection)

## Checklist Before Submission
- [ ] Zoning confirmation letter or approved variation
- [ ] Structural engineer stamp
- [ ] MEP coordination complete
- [ ] Energy compliance (IECC)
- [ ] Accessibility compliance (ICC A117.1)
- [ ] Fire separation / egress verification
- [ ] Kuan final review and stamp

## This Is The Moat
The self-cert license is SHB's single biggest competitive advantage.
Every project that uses it saves months of timeline = months of carry cost = higher returns.
Protect it. Never rush a self-cert stamp. The license is worth more than any single project.
