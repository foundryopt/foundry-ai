## Pro Forma Stages
| Stage | When | Accuracy |
|-------|------|----------|
| Preliminary | Site selection | ±20% |
| SD (Schematic Design) | After SD complete | ±15% |
| CD (Construction Docs) | After CDs / GMP | ±5% |
| As-Built | Project closeout | Actual |

## Key Metrics
- Developer margin target: ≥20%
- Construction cost per SF: track against Chicago market
- Carry cost per month: minimize via self-cert speed
- Cash-on-cash to investors: per fund docs
