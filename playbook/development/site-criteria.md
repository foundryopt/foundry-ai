# Site Selection Criteria

## Must-Have
- [ ] Chicago — target neighborhoods: Lincoln Park, Lakeview, Ravenswood, North Center, Logan Square
- [ ] Lot size supports 4+ units under target zoning
- [ ] Within 0.5mi of CTA rail or high-frequency bus
- [ ] No environmental red flags (Phase I clear or manageable)
- [ ] Acquisition price allows 20%+ developer margin at conservative exit values

## Scoring Matrix

| Factor | Weight | Score (1-5) |
|--------|--------|-------------|
| Zoning (as-of-right vs. rezone needed) | 25% | |
| Acquisition price vs. comps | 25% | |
| Unit count potential | 20% | |
| Neighborhood trajectory | 15% | |
| Construction access / site conditions | 15% | |

## Go / No-Go
- Score ≥ 3.5 → Proceed to underwriting
- Score 2.5–3.5 → Review with Kuan
- Score < 2.5 → Pass

## Data Sources
- CoStar / LoopNet for listings
- Chicago Zoning Map (secondcityzoning.org)
- Cook County Assessor for tax/lot data
- Redfin/Zillow for exit comp validation
