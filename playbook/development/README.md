# Development Playbook

From deal sourcing through acquisition, entitlement, and exit strategy.

1. **Site Identification** → `site-criteria.md`
2. **Underwriting** → `underwriting-checklist.md`  
3. **Acquisition** → `acquisition-flow.md`
4. **Entitlement / Zoning** → `zoning-playbook.md`
5. **Exit Strategy** → `exit-matrix.md`

## SHB Sweet Spot
- Chicago multi-family residential (4–8 units)
- Transit-oriented (within 0.5mi of CTA)
- Zoning: RT-4 or RM-5 (or rezone candidates)
- All-in budget: $2M–$4M per project
- Target: 3–4 concurrent projects at staggered phases
